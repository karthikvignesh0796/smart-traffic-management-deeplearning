# Smart Traffic Management System Using Deep Learning

A real-time traffic monitoring and signal control system built using YOLOv8 and OpenCV.

## Overview
This project detects vehicles from live video streams, measures traffic density, and dynamically adjusts signal timings based on congestion levels. It also includes an Emergency Vehicle Priority Mode that automatically detects ambulances, fire trucks, and police vehicles and triggers green signals to reduce response time.

## Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Deep Learning / Computer Vision

## Features
- Real-time vehicle detection from video feed
- Dynamic traffic signal control based on density
- Emergency vehicle detection and priority signalling
- Congestion pattern analysis

## How to Run
```bash
pip install ultralytics opencv-python
python main.py
```
